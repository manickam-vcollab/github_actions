Translation
============

The usage of VMoveJT is to translate JT model into a CAX file. This tutorial describes the procedure that is involved in the translation. 


  #. Go to Start | Programs | VCollab | VMoveJT. It launches the application as follows

     
     |image1|


  #. Select the **Input File.**

  #. Select the **Output File.** 

  #. Click the **Translate** button to start the JT to CAX conversion.

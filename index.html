VCollab Suite is the next generation software solution developed by Visual Collaboration TechnologiesInc., to provide a unified way to visualize, share, present, document and collaborate CAD and CAE solution data. At the core of the VCollab Suite lies the VCollab proprietary file format named CAx. VCollab Suite comes with products which provide visualization, presentation, documentation and collaboration for CAx files as well as tools for creating CAx files from CAD and CAE solutions.
